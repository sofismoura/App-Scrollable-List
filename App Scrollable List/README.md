# Affirmations App - Lista Rolável com Jetpack Compose

Aplicativo Android que exibe uma lista rolável de 10 cards com afirmações positivas, desenvolvido com Jetpack Compose.

![Screenshot do App](print-card_rolavel.png)

## Sobre o Projeto

O app Affirmations contém uma lista rolável com 10 cards, cada um apresentando uma mensagem motivacional acompanhada de uma imagem inspiradora.

## Funcionalidades

- Lista rolável usando `LazyColumn`
- Cards com imagens e textos
- Design Material 3
- Interface responsiva e moderna

## Pré-requisitos

Para executar este projeto, você precisa ter:

- Familiaridade com listas em Kotlin
- Experiência construindo layouts com Jetpack Compose
- Android Studio instalado
- Conhecimento básico de desenvolvimento Android

## Como Executar

1. Instale o Android Studio (caso ainda não tenha)
2. Clone este repositório:

```bash
git clone https://github.com/joaoxaviersilva/App-Card-Rolavel.git
```

4. Abra o projeto no Android Studio
5. Aguarde o Gradle sincronizar as dependências
6. Execute o app em um emulador ou dispositivo físico

## Tecnologias Utilizadas

- Kotlin
- Jetpack Compose
- Material Design 3
- Android SDK
